## ADDED Requirements

### Requirement: 优惠券叠加配置
系统 SHALL 允许管理员为每个优惠券模板配置是否允许与其他优惠券叠加使用，并在优惠券查询响应中返回该配置。

#### Scenario: 创建允许叠加的优惠券
- **WHEN** 管理员创建优惠券并将允许叠加设置为 true
- **THEN** 系统保存该优惠券的叠加配置为允许
- **AND** 管理端和买家端优惠券响应均返回该配置

#### Scenario: 创建未显式设置叠加配置的优惠券
- **WHEN** 管理员创建优惠券但未提交叠加配置
- **THEN** 系统默认该优惠券不允许叠加使用

#### Scenario: 编辑优惠券叠加配置
- **WHEN** 管理员编辑未被删除的优惠券并修改叠加配置
- **THEN** 系统保存新的叠加配置
- **AND** 已创建订单中的优惠券使用快照不受影响

## MODIFIED Requirements

### Requirement: 管理员发放优惠券
系统 SHALL 允许管理员创建平台级满减优惠券，并配置名称、说明、适用人群、满减门槛、抵扣金额、发放总量、每人领取上限、有效期、是否允许叠加使用和启停状态。

#### Scenario: 创建优惠券成功
- **WHEN** 管理员提交名称、满减门槛、抵扣金额、发放总量、每人领取上限、有效期和叠加配置
- **THEN** 系统创建优惠券活动并返回优惠券 ID
- **AND** 优惠券初始已领取数量为 0
- **AND** 系统保存该优惠券是否允许叠加使用

#### Scenario: 创建无效金额优惠券
- **WHEN** 管理员提交抵扣金额小于等于 0、门槛金额小于 0 或抵扣金额大于门槛金额的优惠券
- **THEN** 系统返回 400 状态码
- **AND** 系统不创建优惠券活动

#### Scenario: 停用优惠券
- **WHEN** 管理员停用某个优惠券活动
- **THEN** 系统将该优惠券状态更新为 `INACTIVE`
- **AND** 买家不能继续领取或使用该优惠券

#### Scenario: 非管理员访问发放接口
- **WHEN** 非管理员用户访问优惠券发放或启停接口
- **THEN** 系统返回 403 状态码

### Requirement: 优惠券使用资格校验
系统 SHALL 在下单使用优惠券前校验买家优惠券归属、状态、有效期、活动状态、订单金额门槛和叠加组合规则。

#### Scenario: 单张可用优惠券通过校验
- **WHEN** 买家使用自己的一张 `UNUSED` 优惠券下单
- **AND** 优惠券活动为 `ACTIVE`、未过有效期且订单商品原始金额达到满减门槛
- **THEN** 系统允许订单使用该优惠券

#### Scenario: 普通券与会员券允许叠加
- **WHEN** 买家同时使用自己的一张普通优惠券和一张会员专属优惠券下单
- **AND** 两张优惠券均为 `UNUSED`、活动为 `ACTIVE`、未过有效期、订单商品原始金额均达到满减门槛
- **AND** 两张优惠券都配置为允许叠加使用
- **THEN** 系统允许订单同时使用两张优惠券

#### Scenario: 任一优惠券不允许叠加
- **WHEN** 买家同时使用普通优惠券和会员专属优惠券下单
- **AND** 任一优惠券配置为不允许叠加使用
- **THEN** 系统返回 400 状态码
- **AND** 系统不创建订单、不扣减库存、不清空购物车、不标记任何优惠券为已使用

#### Scenario: 两张同类型优惠券不可叠加
- **WHEN** 买家同时使用两张普通优惠券或两张会员专属优惠券下单
- **THEN** 系统返回 400 状态码
- **AND** 系统不创建订单、不标记任何优惠券为已使用

#### Scenario: 使用他人优惠券
- **WHEN** 买家使用不属于自己的买家优惠券下单
- **THEN** 系统返回 403 状态码
- **AND** 系统不创建订单、不扣减库存、不清空购物车、不标记优惠券为已使用

#### Scenario: 使用不存在优惠券
- **WHEN** 买家使用不存在的买家优惠券 ID 下单
- **THEN** 系统返回 404 状态码
- **AND** 系统不创建订单、不扣减库存、不清空购物车

#### Scenario: 使用已使用优惠券
- **WHEN** 买家使用状态为 `USED` 的优惠券下单
- **THEN** 系统返回 400 状态码
- **AND** 系统不创建订单、不扣减库存、不清空购物车

#### Scenario: 订单未达到满减门槛
- **WHEN** 买家使用优惠券下单但订单商品原始金额小于任一优惠券满减门槛
- **THEN** 系统返回 400 状态码
- **AND** 系统不创建订单、不扣减库存、不清空购物车、不标记优惠券为已使用
