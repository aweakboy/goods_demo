## MODIFIED Requirements

### Requirement: 查看订单详情
系统 SHALL 允许管理员查看任意订单的完整详情，包括订单基础信息、收货地址、商品条目、支付/订单状态、物流摘要和物流轨迹。

#### Scenario: 查看订单详情
- **WHEN** 管理员点击某笔订单查看详情
- **THEN** 系统返回订单完整信息：订单ID、买家信息、收货地址、所有商品条目（名称/单价/数量）、总金额、状态、物流公司、物流单号、物流状态、物流轨迹、创建时间

#### Scenario: 查看物流异常订单详情
- **WHEN** 管理员查看 shipment 状态为 EXCEPTION 的订单详情
- **THEN** 系统返回异常状态、异常原因轨迹和最近更新时间

#### Scenario: 查看历史物流订单详情
- **WHEN** 管理员查看没有 shipment 但有旧物流单号的订单详情
- **THEN** 系统展示旧物流单号，并标记暂无结构化物流轨迹
