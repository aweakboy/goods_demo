## ADDED Requirements

### Requirement: 店铺主页地图展示
系统 SHALL 在店铺地址已校验且有经纬度时，在店铺主页展示店铺位置地图。

#### Scenario: 店铺主页展示地图
- **WHEN** 访客访问有效店铺主页
- **AND** 店铺地址校验状态为 `VALID` 且存在经纬度
- **THEN** 页面展示店铺地址文本
- **AND** 页面展示以店铺坐标为中心的地图 marker

#### Scenario: 店铺主页无经纬度时降级
- **WHEN** 访客访问店铺主页但店铺地址未校验或缺少经纬度
- **THEN** 页面继续展示店铺地址文本
- **AND** 页面不展示地图 marker

#### Scenario: 地图 SDK 加载失败
- **WHEN** 访客访问店铺主页且地图 SDK 加载失败
- **THEN** 页面展示地图不可用提示
- **AND** 页面继续展示店铺地址文本和商品列表
