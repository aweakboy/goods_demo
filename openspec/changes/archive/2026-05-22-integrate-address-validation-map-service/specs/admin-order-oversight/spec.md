## ADDED Requirements

### Requirement: 管理员订单收货地图展示
系统 SHALL 在管理员订单详情中，对已校验且有经纬度的收货地址展示地图定位，帮助管理员核查订单配送信息。

#### Scenario: 管理员订单详情展示收货地图
- **WHEN** 管理员查看订单详情
- **AND** 订单收货地址快照包含已校验经纬度
- **THEN** 前端展示收货地址文本
- **AND** 前端展示收货地址地图 marker

#### Scenario: 管理员订单详情无经纬度时降级
- **WHEN** 管理员查看订单详情但订单收货地址缺少经纬度或未校验
- **THEN** 前端只展示文本收货地址
- **AND** 前端不展示地图 marker
